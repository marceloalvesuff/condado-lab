Reportagem sobre a política do TikTok de disponibilizar API gratuita apenas para pesquisadores de universidades europeias e norte-americanas, excluindo pesquisadores do Sul Global como os do Brasil. Marcelo Alves comenta as implicações dessa assimetria para a pesquisa sobre desinformação no contexto brasileiro.
